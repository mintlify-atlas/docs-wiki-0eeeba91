This Overview group is the entry point for engineers reading the ABP framework source. ABP (sometimes written `abp` in code, namespaced under `Volo.Abp.*`) is a modular ASP.NET Core based application framework whose code lives in the `abpframework/abp` monoŌĆærepo.
